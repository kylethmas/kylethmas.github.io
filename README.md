# Kyle Thomas – Personal Portfolio

A minimalist, single-page portfolio to showcase my work as a cloud & DevOps-focused software engineer.

## Stack

- HTML5
- Modern CSS (no framework)
- Vanilla JavaScript (for mobile nav + footer year)

## Structure

- `index.html` – main page (hero, experience, projects, skills, contact)
- `css/style.css` – layout, theme and components
- `img/` – social icons and favicon
- `Kyle-Thomas-CV.pdf` – downloadable CV (update the filename/path if needed)

## Running Locally

No build step required.

# Option 1: just open
open index.html
